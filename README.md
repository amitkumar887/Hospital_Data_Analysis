🏥 Hospital Management Dashboard

An interactive and visually rich Hospital Management Dashboard built using Power BI to analyze and monitor hospital operations efficiently.
This dashboard provides insights into Patients, Doctors, Finance, Medicines, Appointments, Billing, and Hospital Performance through dynamic visualizations and KPI tracking.

🚀 Project Overview

The Hospital Dashboard is designed to help hospitals and healthcare organizations:

Track patient information
Monitor doctor performance
Analyze financial data
Manage medicine stock & sales
View appointments and reports
Improve decision-making using data visualization

The project contains multiple dashboard pages with interactive charts, slicers, KPIs, and reports.
